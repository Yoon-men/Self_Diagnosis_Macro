## What the fuck is this?
가상 키패드를 찢어발기는 자가진단 매크로 프로그램이오.<br><br> 
자가진단 사이트가 업데이트를 계속하는 한, 창과 방패의 싸움은 계속될 것임을 약속하오.<br><br>
허나, 이 처절한 전투의 본 목적을 잊어버려서는 안되는 법.<br><br>
**부디 그대의 건강에 이상이 없을 때만 이용해주기를 바랄 뿐이오.**<br><br>
